The Legend of Zelda NES remake:

I worked on Link's mechanics (movement, attack, weapons), the level design/enemy placement, as well as a few enemies (wallmaster, bladetraps). I also worked on the custom level design and dash mechanic, and a portion of the audio and visuals. The project was done using Unity in C#.
The Windows and MacOS builds are available here: The Legend of Zelda NES remake

Orbital Mechanics:

I made an original puzzle platforming physics game that uses real orbital mechanics in a fun way. If you want to gain an intuitive understanding of orbital mechanics that even the Gemini 4 spaceflight team hadn't figured out, play this game! The project was done using Unity in C#.
The Windows and MacOS builds are available here: Orbital Mechanics
